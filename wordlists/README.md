# APDE Wordlists

This directory contains parameter wordlists used by APDE scanner.

## common.txt

General-purpose parameter names for:

- GET parameter discovery
- POST parameter fuzzing
- JSON key discovery
- Header injection testing

Focus areas:
- Authentication parameters
- Redirect behavior
- Debug features
- File handling
- Pagination & filtering
- API-related keys

---

⚠️ Note:
These wordlists are designed for behavioral discovery,
not brute-force attacks.

Use responsibly and only on authorized targets.
